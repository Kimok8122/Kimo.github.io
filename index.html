Google Photos for Audio

The Google Photos for Audio app essentially took the idea of Google Photos, which runs a machine learning algorithm on the images so that users are able to search through the images with objects, and applied this to audio files. Working in a group and using Agile development methodologies, the app was created with a group. The front end of the application was built using React and the backend was built using Django and AWS.
